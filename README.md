Descripción
Arbor es un proyecto web sencillo desarrollado con HTML, CSS y JavaScript. Su objetivo es mostrar una página básica con diseño limpio, título personalizado y una estructura organizada, ideal para aprender los fundamentos del desarrollo web y el uso de Git y GitHub.

- CSS3
- JavaScript (Vanilla)
- Git & GitHub

arborea v.3/

Características
- Diseño simple y responsivo
- Interfaz limpia y fácil de entender
- Separación de estructura, estilos y lógica
- Proyecto ideal para práctica de Git y GitHub
